**Overview**
This repo contains Excel VBA tools designed to automate reporting for a Course Management System. It cleans raw data, enriches it with calculations, and generates analytical dashboards.

**Key Features**
Data Prep & Enrichment: Automates calculation of student ages, course counts, and formats data for analysis.

Pivot Table Generation: Creates and updates three distinct analytical views:

Enrollments by Location and Status.

Enrollments by Schedule and Location.

Monthly Attendance Percentage.

Filtering & Export: Includes tools to filter enrollment data by specific date ranges and export those snapshots to new worksheets.
